# Calorie Counter 

## Project Overview

This is a Calorie Counter web application that allows users to add food items and track their daily calorie consumption. The application calculates total calories, displays a progress bar based on a daily goal, and stores data in localStorage so that information persists after page refresh.

---

## Features

- Add food items with calorie tracking
- Fetch calorie data from a local JSON database
- Manual calorie input for unknown foods
- Display list of all added food items
- Delete food items from the list
- Reset all data for a new day
- Calculate total calories consumed
- Display progress bar for calorie tracking
- Persist data using localStorage

---

## How to Use the App

1. Open the project using Live Server or any local server.
2. Enter a food name in the input field.
3. If the food exists in the database, calories are automatically assigned.
4. If the food is not found, manually enter the calorie value.
5. Add the food to your daily list.
6. View total calories and progress bar update in real time.
7. Reset the app to start a new day.
